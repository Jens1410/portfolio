# Personal Portfolio – Jens Ackermann

This repository contains the source files for my personal portfolio, built with **MkDocs** and the **Material for MkDocs** theme.  
The site presents my professional profile as a technical writer and information architect, including skills, methods, and selected examples of my work.

## 📚 About the Project
The portfolio is designed as a clean, modular documentation site.  
It focuses on:

- technical writing and information architecture  
- analytical and explanatory skills  
- documentation workflows and tooling  
- selected project insights  

The site is generated from Markdown files and custom CSS.

## 🛠 Technology Stack
- **MkDocs** – static site generator  
- **Material for MkDocs** – theme and UI components  
- **Markdown** – content  
- **Custom CSS** – layout refinements  

## 🚀 Deployment
The site is deployed via **GitHub Pages** using the `gh-pages` branch generated by:
mkdocs gh-deploy
