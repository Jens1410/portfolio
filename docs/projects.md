# Projekte

Eine Auswahl von Projekten, in denen ich moderne Dokumentations-Workflows aufgebaut, Inhalte strukturiert und technische Themen klar und verständlich aufbereitet habe.  
Die Beispiele zeigen meinen Fokus auf Modularität, Informationsarchitektur und effiziente Tools.

## Dokumentationsprojekte

### Aufbau eines modularen Dokumentationssystems

!!! abstract "Rolle: Technical Writer & Information Architect"
    **Ziel:** Aufbau einer klar strukturierten, modularen Dokumentation für ein technisches Produkt.  
    **Ergebnis:** Eine wartbare, erweiterbare Dokumentationsarchitektur, die Inhalte konsistent und nachvollziehbar macht.

**Highlights:**

- Analyse bestehender Inhalte und Neuaufbau der Struktur

- Erstellung eines konsistenten Navigations- und Seitenkonzepts

### Migration von Word‑Dokumenten zu Markdown

!!! abstract "Rolle: Technical Writer"

    **Ergebnis:** Klare, versionierbare Dokumentation mit besserer Nachvollziehbarkeit und weniger Redundanz.

**Highlights:**

- Konvertierung und Bereinigung bestehender Inhalte
